Aproximación por Mínimos Cuadrados

Ajuste óptimo de curvas a datos discretos en el sentido de la norma cuadrática

Profe W

Sección 01El Problema de Aproximación Discreta

Dado un conjunto de \(m\) puntos medidos experimentalmente \((x_1,y_1),\ldots,(x_m,y_m)\) con \(x_i \neq x_j\), queremos encontrar la ecuación de la curva \(y = f(x)\) que mejor representa el comportamiento de esos datos, sin necesariamente pasar por todos ellos.

💡
Intuición

Interpolación obliga a que la curva pase exactamente por todos los puntos. El ajuste por mínimos cuadrados, en cambio, busca una curva que minimice el error total respecto a todos los puntos. Esto es crucial cuando los datos contienen ruido experimental.

Medidas de Error

Para cuantificar qué tan bien se ajusta \(f(x)\) a los datos, definimos la desviación o residuo en cada punto como \(e_k = f(x_k) - y_k\). A partir de estos residuos, existen varias formas de medir el error global:

E∞(f)
\(\max_{1\le k\le m}|f(x_k)-y_k|\)
Error máximo — sensible a datos atípicos
E₁(f)
\(\dfrac{1}{m}\sum_{k=1}^{m}|f(x_k)-y_k|\)
Error medio — más robusto
E₂(f) ✓
\(\sqrt{\dfrac{1}{m}\sum_{k=1}^{m}(f(x_k)-y_k)^2}\)
Error cuadrático medio — el más utilizado
¿Por qué el error cuadrático?

Los errores \(E_\infty(f)\) y \(E_1(f)\) no son diferenciables en general, lo que complica la optimización. Minimizar \(E_2(f)\) es equivalente a minimizar la suma de cuadrados:

$$E(f) = \sum_{k=1}^{m}\bigl(f(x_k) - y_k\bigr)^2$$

Esta función sí es diferenciable y conduce a un sistema lineal de ecuaciones normales, resoluble de forma directa. Por eso el método se llama mínimos cuadrados.

Error conceptual frecuente

Los residuos \(e_k = f(x_k) - y_k\) se miden en dirección vertical (diferencia en \(y\)), no como distancia perpendicular a la curva. El método de mínimos cuadrados minimiza la suma de cuadrados de estas distancias verticales.

¿Cuándo usar mínimos cuadrados?

📊
Datos ruidosos
Mediciones con error experimental donde no conviene interpolar exactamente.
📈
Tendencias
Se quiere capturar el comportamiento global sin reproducir fluctuaciones.
🔢
Más datos que parámetros
Cuando \(m \gg n+1\), el sistema está sobredeterminado y no tiene solución exacta.
🎯
Modelo conocido
Cuando la forma funcional del modelo (\(f = Ax+B\), \(f = ae^{bx}\), etc.) es conocida a priori.

Sección 02Recta de Regresión

El caso más simple y fundamental es encontrar la recta \(y = Ax + B\) que mejor se ajusta a \(m\) puntos en el sentido de mínimos cuadrados.

Definición — Recta de Regresión

Dados \(m\) puntos \((x_1,y_1),\ldots,(x_m,y_m)\) con \(x_i \neq x_j\), la recta de regresión u óptima en el sentido de mínimos cuadrados es \(y = Ax + B\) donde \(A\) y \(B\) minimizan la función de error:

$$E(A,B) = \sum_{k=1}^{m}(Ax_k + B - y_k)^2$$

Derivación del Sistema Normal

Para hallar el mínimo de \(E(A,B)\) igualamos a cero sus derivadas parciales:

$$\frac{\partial E}{\partial A} = 2\sum_{k=1}^{m}x_k(Ax_k+B-y_k) = 0$$ $$\frac{\partial E}{\partial B} = 2\sum_{k=1}^{m}(Ax_k+B-y_k) = 0$$

Expandiendo y reorganizando, obtenemos el sistema de ecuaciones normales:

Sistema de Ecuaciones Normales (n = 1)
$$\begin{cases} A\displaystyle\sum_{k=1}^{m}x_k^2 + B\displaystyle\sum_{k=1}^{m}x_k = \displaystyle\sum_{k=1}^{m}x_k y_k \\[10pt] A\displaystyle\sum_{k=1}^{m}x_k + Bm = \displaystyle\sum_{k=1}^{m}y_k \end{cases}$$

La solución \((A^*, B^*)\) de este sistema garantiza el mínimo global de \(E(A,B)\), ya que la función es convexa.

⚠️
Límite de la recta de regresión

La recta de regresión es óptima entre todas las rectas, pero si los datos siguen un patrón no lineal (parabólico, exponencial, logístico…), el error seguirá siendo grande. En ese caso, conviene un ajuste polinomial o no lineal.

Sección 03Ajuste Polinomial

Cuando los datos muestran curvatura, la recta de regresión no es suficiente. Podemos buscar el polinomio de grado \(n\) óptimo en el sentido de mínimos cuadrados.

Caso Cúbico — Derivación

Para \(f(x) = Ax^3 + Bx^2 + Cx + D\), la función de error es:

$$E(A,B,C,D) = \sum_{k=1}^{m}(Ax_k^3 + Bx_k^2 + Cx_k + D - y_k)^2$$

Igualando a cero las cuatro derivadas parciales, se obtiene el sistema de ecuaciones normales \(4 \times 4\).

Caso General — Polinomio de Grado n

Sistema Normal General

El polinomio \(y = a_n x^n + \cdots + a_1 x + a_0\) óptimo en el sentido de mínimos cuadrados se obtiene resolviendo el sistema \((n+1)\times(n+1)\):

$$\begin{pmatrix} \sum x_k^{2n} & \sum x_k^{2n-1} & \cdots & \sum x_k^n \\ \sum x_k^{2n-1} & \sum x_k^{2n-2} & \cdots & \sum x_k^{n-1} \\ \vdots & \vdots & \ddots & \vdots \\ \sum x_k^n & \sum x_k^{n-1} & \cdots & m \end{pmatrix} \begin{pmatrix}a_n \\ a_{n-1} \\ \vdots \\ a_0\end{pmatrix} = \begin{pmatrix}\sum x_k^n y_k \\ \sum x_k^{n-1} y_k \\ \vdots \\ \sum y_k\end{pmatrix}$$

donde todas las sumas son de \(k = 1\) a \(m\). La matriz del sistema se llama matriz de Gram y siempre es simétrica y definida positiva si los \(x_k\) son distintos.

Ejemplo — Polinomio de grado 2

Ajustar un polinomio cuadrático \(p_2(x) = a_2 x^2 + a_1 x + a_0\) a los datos \(\{(x_k,y_k)\}_{k=1}^{10}\) con \(x_k = 0.1k\), \(y_k = x_k + \cos(\sqrt{k})\).

Ver solución y comparación con la recta

El sistema normal de \(3 \times 3\) resulta en:

$$\begin{pmatrix}3.5033 & 3.85 & 3.025\\ 3.85 & 3.025 & 3.5 \\ 3.025 & 3.5 & 10\end{pmatrix} \begin{pmatrix}a_2 \\ a_1 \\ a_0\end{pmatrix} = \begin{pmatrix}-0.3176 \\ -0.3176 \\ 0.4117\end{pmatrix}$$

Solución: \(a_2 \approx 2.3207\), \(a_1 \approx -3.2122\), \(a_0 \approx 0.9144\)

$$p_2(x) \approx 2.3207x^2 - 3.2122x + 0.9144$$
ModeloError de mínimos cuadrados
Recta \(y = -0.6594x + 0.4039\)\(\approx 0.2867\)
Parábola \(p_2(x)\)\(\approx 0.0027\)

La parábola reduce el error en más de 100 veces respecto a la recta. Esto confirma que el comportamiento de los datos es cuadrático.

📐
¿Qué grado elegir?

Un grado alto no siempre es mejor — puede producir sobreajuste (overfitting). La regla práctica: aumentar el grado mientras el error cuadrático disminuya significativamente, pero nunca usar \(n \ge m\) (eso sería interpolación).

Pasos para el ajuste polinomial

  • 1
    Definir el grado \(n\) del polinomio basándose en la gráfica de los datos.
  • 2
    Calcular las sumas \(\sum x_k^j\) para \(j=0,1,\ldots,2n\) y \(\sum x_k^j y_k\) para \(j=0,1,\ldots,n\).
  • 3
    Armar la matriz de Gram (simétrica) y el vector independiente.
  • 4
    Resolver el sistema lineal \((n+1)\times(n+1)\) (Gauss, LU, etc.).
  • 5
    Calcular el error cuadrático \(E = \sum (p_n(x_k)-y_k)^2\) para validar.

Sección 04Ajuste No Lineal y Linealización

Muchos fenómenos reales siguen patrones no lineales: crecimiento exponencial, leyes de potencia, curvas logísticas. Para aplicar mínimos cuadrados a estos modelos, se puede realizar un cambio de variables que linealice el problema.

Advertencia importante

La curva obtenida tras linealizar y aplicar mínimos cuadrados no es la óptima para el problema original — es óptima para el problema transformado. Para la óptima real se requiere optimización no lineal. Aun así, la linealización da una buena aproximación inicial.

Tabla de Linealizaciones Usuales

Modelo original \(y = f(x)\)Transformación \(Y = F(X)\)Cambio de variables
\(y = C e^{Ax}\)\(Y = a_0 + a_1 X\)\(X = x,\; Y = \ln y,\; C = e^{a_0},\; A = a_1\)
\(y = C x^A\)\(Y = a_0 + a_1 X\)\(X = \ln x,\; Y = \ln y,\; C = e^{a_0},\; A = a_1\)
\(y = \dfrac{1}{Cx+A}\)\(Y = a_0 + a_1 X\)\(X = x,\; Y = \dfrac{1}{y},\; C = a_1,\; A = a_0\)
\(y = \dfrac{A}{x} + C\)\(Y = a_0 + a_1 X\)\(X = \dfrac{1}{x},\; Y = y,\; C = a_0,\; A = a_1\)
\(y = \dfrac{L}{1+Ce^{At}}\) (logística)\(Y = a_0 + a_1 X\)\(X = t,\; Y = \ln\!\left(\tfrac{L}{y}-1\right),\; C = e^{a_0},\; A = a_1\)
